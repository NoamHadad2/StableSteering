# Output Folder

This folder contains generated reports and optional release artifacts.

## What may appear here

- Playwright HTML reports
- Playwright traces, screenshots, and videos
- packaged release zip files under `output/releases/`
- real example bundles under `output/examples/`

Example:

- `output/examples/real_e2e_example_run/`
  Contains a real GPU-backed walkthrough with generated images, manifest, and HTML reports.

Tracked sample bundle:

- [real_e2e_example_run.html](./examples/real_e2e_example_run/real_e2e_example_run.html)
- [session_trace_report.html](./examples/real_e2e_example_run/session_trace_report.html)
- [manifest.json](./examples/real_e2e_example_run/manifest.json)

Generated by:

- `python scripts/create_real_e2e_example.py`

## Safe cleanup

This folder can be deleted safely. It only contains generated output and can be recreated by the app, test tools, or release scripts.

Human Pairwise Evaluation Package

This directory packages a small pairwise human-evaluation pilot around the current paper artifacts.

Included materials

  • curated pairs: 6
  • blank annotation sheet: annotations_blank.csv
  • pair manifest: pairs.csv
  • browser-friendly preview: pairwise_review.html

Collection rule

Judges should answer: Which image better satisfies the prompt while remaining visually coherent?

Allowed responses:

  • left
  • right
  • tie
  • invalid

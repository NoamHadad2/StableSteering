# Reproducibility Checklist

This checklist captures what the current paper package already supports and what is still missing for a stronger research artifact.

## Already supported

- runnable local app
- real GPU-backed Diffusers backend
- per-session YAML configuration
- preserved replay and trace artifacts
- checked-in qualitative case-study bundle
- backend and browser verification
- locked minimal-baseline protocol bundle
- executed minimal-baseline matrix bundle
- runner for the baseline-matrix package
- paper-facing results directory for the baseline matrix

## Not yet complete as a research package

- repeated-seed robustness checks
- analysis notebook or summary script for paper tables
- broader comparative baseline matrix
- fuller bibliography for systems/observability positioning

## Safe current paper claim

StableSteering is reproducible as a software artifact and traceable as an interaction system. It now includes a bounded minimal comparative bundle, but it is not yet a complete comparative research package.
